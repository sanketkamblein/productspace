# Sanket Kamble Portfolio - GitHub Pages

This is a static HTML/CSS/JavaScript portfolio website designed for GitHub Pages deployment.

## Setup Instructions

1. **Clone or download this repository**

2. **Add your images:**
   - Create an `images` folder in the root directory
   - Add `sanket-portrait.png` and `sanket-realistic.jpg` to the images folder
   - Create a `files` folder and add `sanket-kamble-resume.pdf`

3. **Update project images:**
   - Replace the placeholder image URLs in `script.js` with your actual project images
   - You can either use local images or external URLs

4. **Deploy to GitHub Pages:**
   - Push the code to a GitHub repository
   - Go to repository Settings > Pages
   - Select "Deploy from a branch" and choose "main" branch
   - Your site will be available at `https://yourusername.github.io/repository-name`

## Features

- Responsive design that works on desktop, tablet, and mobile
- Interactive animated timeline (desktop only)
- Project modal with detailed information
- Smooth scrolling navigation
- Professional styling with purple and gold color scheme

## Customization

- Update project data in the `projects` array in `script.js`
- Modify colors and styling in `styles.css`
- Add or remove sections by editing `index.html`

## Browser Support

This portfolio works in all modern browsers and is optimized for GitHub Pages hosting.
